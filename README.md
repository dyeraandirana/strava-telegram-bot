# Strava Telegram Bot

Bot Telegram yang terhubung ke akun Strava pengguna. Setelah login, bot akan membaca aktivitas olahraga terbaru dan memberikan analisis langsung di Telegram.

## Fitur
- Autentikasi Strava via OAuth
- Penyimpanan token di Google Sheets
- Analisis aktivitas olahraga
- Respon otomatis di Telegram

## Teknologi
- Python
- Flask
- Telegram Bot API
- Strava API
- Google Sheets
